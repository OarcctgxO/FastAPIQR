HTTP-GET сервер на FastAPI для создания простых QR-кодов.

Принимает любую строку текста с длиной от 1 до 1000 символов, возвращает png картинку с QR-кодом.
Использует минимальную коррекцию и динамически меняющуюся для текста версию QR (версия QR простым языком - это "сложность" картинки).
Использует PyPNGImage для создания кодов.

По сути, я просто портировал свой другой проект qr_code_server в FastAPI, но поскольку все решения асинхронного сервера FastAPI реализует под капотом (кроме UDP-discover (использую в локальной сети), что не нужно в большом продакшене из-за DNS-ресолва), то и от оригинального проекта в этом осталась только функция генерации png картинки.
Поскольку в оригинале я использовал freethreaded интерпретатор python3.14, то и сохранял картинки без Pillow, что перекочевало и сюда.
